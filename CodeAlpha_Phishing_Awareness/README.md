# Phishing Awareness Training Website

An interactive cybersecurity awareness website developed using HTML, CSS, and JavaScript to educate users about phishing attacks, fake websites, and social engineering techniques.

</div>

---

## Overview

This project is a Phishing Awareness Training Website designed to educate users about common phishing attacks and cybersecurity threats. The website explains how attackers use phishing emails, fake websites, and social engineering tactics to steal sensitive information.

The project includes interactive educational content, colorful awareness sections, real-world phishing examples, and 10 interactive quiz questions to improve user engagement and cybersecurity awareness.

---

## Features

- Modern and responsive website design
- Interactive phishing awareness training module
- Recognition of phishing emails and fake websites
- Social engineering awareness content
- Cybersecurity best practices and prevention tips
- Real-world phishing attack examples
- 10 interactive quiz questions with instant feedback
- Colorful content sections and animations
- Beginner-friendly user interface
